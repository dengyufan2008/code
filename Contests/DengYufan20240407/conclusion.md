## T2([b](https://www.luogu.com.cn/problem/U415121)) [计数 Dp] [容斥] [多项式]

期望:65 实际:65 难度:3 用时:2.5h

同 HNOI Day1 T3 (wormhole), Dp 方程列出来后可以考虑其的实际意义来优化.

多项式多点求值: 给定 $n$ 项多项式 $F(z) = \sum_i f_i z^i$ 与长为 $m$ 的序列 $a$, 求 $F(a_i)$. 考虑 $F(z) = F_i(z) * (z - a_i) + ans_i$, 则只要对每个 $i$ 求出 $F(z) \bmod (z - a_i)$ 的值即可. 考虑分治取模, 左边每次模 $\prod_{i = l}^{mid} (z - a_i)$, 右边同理. $O(n \log^2 n)$ ~~或许可以利用 class 20240404 转置原理, 但我还不会.~~
