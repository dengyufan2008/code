## T1(pain) [搜索] [合并等价类] [狄利克雷卷积]

期望:22 实际:22 难度:2 用时:2h

如下代码:
```c++
for (int i = 1; i <= n; i++) {
  for (int j = i + i; j <= n; j += i) {
    f[j] -= f[i];
  }
}
```
其本质上为将 $f$ 卷上 $\mu$.

事实上, $n$ 以内的本质不同 质因数分解的指数集合 是极少的, 当 $n = 10^{18}$ 时为 $10^4$ 量级.
