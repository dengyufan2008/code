# 选课及评价

表中排列的顺序为对选择该门课程的推荐程度. 打不过体育课的就可以拉倒了.

度量的值域均为 $[1, 5]$. 作业坐牢程度意为重复性的简单工作的量, 或者繁琐的严谨证明, 相信大家都不喜欢. 但是太过困难而导致的坐牢不在考虑范围, 这方面应参考作业难度.

## 预科上 (2025)

### 形式语言与自动机 / 2pts / 40240432-2 / 季铮锋

学习难度:3 作业难度:3 作业坐牢程度:3 期末难度:3 成绩:A

内容有趣, 可以融入 OI 知识.

### 高等微积分(1) / 5pts / 30420095-2 / 王晓峰

学习难度:4 作业难度:5 作业坐牢程度:1 期末难度:4 成绩:B+

王晓峰自编讲义对基础部分的阐述很好. 解题方法主要依靠作业训练.

### 线性代数 / 4pts / 20470044-0 / 徐勇

学习难度:4 作业难度:3 作业坐牢程度:5 期末难度:4 成绩:A-

上课与阅读教材等价. 如果不要 3% 的考勤分就可以翘.

### 体育(1) / 1pts / 10720011-7 / 褚慧龙

学习难度:1 期末难度:5 成绩:C+

清华拳, 认真学会动作每周打一套就能熟练掌握.

但是期末 20% 分数的 3km 长跑有分标准是 18min, 还有人类吗???

期末有 20% 的成绩需要参加集体锻炼, 一周两次每次 1h. 因此体育可以认为一周三节课.

体测 80pts 可以获得下个学期优先选择体育课项目权.

### 计算机科学基础 / 2pts / 10240022-90 / 史元春

学习难度:2 作业难度:1 作业坐牢程度:1 期末难度:1 成绩:A

知识获取途径主要为老师不加解释的介绍名词, 自行查阅 chatgpt.

### 思想道德与法治 / 3pts / 10680053-3 / 戴木才

学习难度:1 作业难度:2 作业坐牢程度:1 期末难度:2 成绩:B+

需要参加两次 (取决于不同的老师) 小班讨论, 但是隔周上课.

### 离散数学(1) / 3pts / 20240013-1 / 马昱春

学习难度:1 作业难度:1 作业坐牢程度:5 期末难度:2 成绩:A-

唯一真史.
