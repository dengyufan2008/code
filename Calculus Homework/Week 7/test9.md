Prob 1. 讨论 $\sqrt{ab}$, $\frac{b-a}{\ln b - \ln a}$, $\frac{a+b}{2}$ 的大小关系.

假设 $0<a<b$, 否则容易看出大小关系. 先证明 $\frac{b-a}{\ln b - \ln a} < \frac{a+b}{2}$. 设 $x=\frac{b}{a}$, 即证明 $\frac{x-1}{\ln x} < \frac{x+1}{2}$. 我们有 $\lim_{x \rightarrow 1} \frac{x-1}{\ln x} = \lim_{x \rightarrow 0} \frac{x}{\ln (x+1)} = \frac{1}{1+o(1)} = 1 = \lim_{x \rightarrow 1} \frac{x+1}{2}$. 设 $f(x) = (\frac{x-1}{\ln x})' = \frac{\ln x + \frac{1}{x} - 1}{\ln^2 x}$. 有 $\lim_{x \rightarrow 1} f(x) = \lim_{x \rightarrow 0} \frac{\ln(x+1) + \frac{1}{x+1} - 1}{\ln^2(x+1)} = \lim_{x \rightarrow 0} \frac{(x-\frac{x^2}{2}+o(x^2)) + (1-x+x^2+o(x^2)) - 1}{x^2+o(x^2)} = \frac{\frac{1}{2}+o(1)}{1+o(1)} = \frac{1}{2} = (\frac{x+1}{2})'$. 又有 $f'(x) = \frac{2x - x\ln x - \ln x - 2}{x^2\ln^3 x}$, 设 $g(x) = -x^2 \ln^3 x f'(x) = x \ln x + \ln x + 2 - 2x$, 显然 $g(1)=0$, $g'(x) = \ln x + \frac{1}{x} - 1$. 由于当 $x > 1$ 时 $\ln x > x-1$, 则 $\ln \frac{1}{x} > 1-x$ 即 $\ln x > 1 - \frac{1}{x}$, 故 $g'(x) > 0$, $g(x) > 0$, $f'(x) < 0$, $f(x) < \frac{1}{2}$, $\frac{x-1}{\ln x} < \frac{x+1}{2}$.

再证明 $\sqrt{ab} < \frac{b-a}{\ln b - \ln a}$, 即 $\sqrt x < \frac{x-1}{\ln x}$, 即 $x-\sqrt x \ln x - 1 > 0$. 显然当 $x=1$ 时取等. 设 $h(x) = 2 \sqrt x \cdot (x-\sqrt x \ln x - 1)' = 2 \sqrt x - \ln x - 2$, 有 $h(1)=0$, $h'(x) = \frac{\sqrt x - 1}{x} > 0$. 故 $h(x) > 0$, $x - \sqrt x \ln x - 1 > 0$, $\sqrt x < \frac{x-1}{\ln x}$.
